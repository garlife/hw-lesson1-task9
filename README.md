# Readme Голосовой список покупок

## About

Забудьте о бумажных блокнотах, утомительных записях перед походом в магазин. 
Составляйте список покупок легко просто надиктовав их в приложение.
Приложение само создаст список и отобразит список в удобном формате чек-боксов.
По мере совершения покупок вы просто отмечатете элементы, которые уже купили.

## Функции

* Создание списка через аудиозапись
* Создание списка вручную
* Просмотр списка
* Редактирование списка: удаление элементов, редактирование названий, состояний
* Удаление списка
* Возможность поделиться списком

## Пример списка

*Название списка №1*

- [ ] Элемент №1 списка
- [ ] Элемент №2 списка
- [X] Отмеченный элемент №3 в списке
