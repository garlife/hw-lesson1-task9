# hw-lesson1-task9 - Голосовой список покупок

## About

Забудьте о бумажных блокнотах, утомительных записях перед походом в магазин. 
Составляйте список покупок и легко просто! Для этого нужно лишь надиктовать их приложению.
Приложение само создаст список и отобразит его в удобном формате чек-боксов.
По мере совершения покупок вы просто отмечатете элементы, которые уже купили.

## Функции

* Создание аудиозаписи
* Распознование речи
* Сохранение распознанных элементов в формате нового списка покупок
* Создание списка вручную
* Просмотр списка
* Редактирование списка: удаление элементов, редактирование названий, состояний
* Удаление списка
* Возможность поделиться списком

## Пример списка

*Название списка покупок №1*

- [ ] Элемент №1 списка
- [ ] Элемент №2 списка
- [X] Отмеченный элемент №3 в списке
